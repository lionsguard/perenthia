During the Alpha and Beta testing phases of the game Perenthia will occasionally go offline to allow new updates to the game or lock game play after features have been tested. With each phase testers will receive emails informing them of when the game will come back online.



What is Perenthia?

erenthia is a text-based adventure game or persistent browser based game that you play over your internet connection through the use of a web browser such as Microsoft Internet Explorer, Mozilla Firefox or Google Chrome. Perenthia is set in a fantasy world with dragons, knights and magic.



Do I need to install anything to play Perenthia?

Yes. You will need to install the Microsoft Silverlight plugin.



While waiting for the game to come back online visit the Official Perenthia Web Site or stop by the Perenthia Forums to learn more about the game.
